# 🥁 Musical Drums

An interactive Simon-style memory game built with HTML, CSS, and JavaScript.

## 🎮 How to play

1. Press any key to start the game
2. Watch the sequence of colored buttons light up and listen to their sounds
3. Repeat the sequence by clicking the buttons in the correct order
4. Each round adds one more step — how far can you go?

## 🛠️ Built with

- HTML5
- CSS3
- JavaScript (ES5)
- jQuery


## 🚀 Getting started

1. Clone the repository:
```
git clone https://github.com/Isadora888/Musical-Drums.git
```
2. Open `index.html` in your browser — no installation needed.
